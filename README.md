pybaidudisk
===========

百度网盘 命令行 Python 版

## example

```
# cloud download
> ./bdisk.py wget http://example.com/file.zip
# file operations
> ./bdisk.py ls
......
> ./bdisk.py ls /Path
......
> ./bdisk.py rename /Path/To/File/FileName NewFileName
......
> ./bdisk.py mv /Path/To/File /Path/To/NewFile
......
> ./bdisk.py du
......
> ./bdisk.py rm /Path/To/File
......
```
